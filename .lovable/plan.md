

# Amadeus Survivor – Web Completa

## Estilo Visual
- Tema oscuro tipo terminal/consola GDS: fondo negro/gris oscuro, texto verde neón, blanco y gris
- Tipografía monoespaciada (JetBrains Mono / Fira Code) para títulos y elementos destacados
- Sans-serif limpia (Inter) para cuerpo de texto
- Animaciones sutiles tipo "typing" en terminal
- Bordes con efecto terminal, cards con estilo consola

## Páginas a Crear

### 1. Home
8 bloques según especificación:
- Hero con título grande estilo terminal, subtítulo y 2 CTAs
- Sección "Qué es Amadeus Survivor" con texto explicativo y lista de público objetivo
- 3 tarjetas de módulos (Vuelos, Hoteles, Coches) con descripción, botón "Ver más" y botón "Comprar"
- Sección "Cómo funciona" con pasos visuales (iconos + texto)
- Sección "Para quién es" con lista estilizada
- Espacio para vídeo embebido (YouTube iframe placeholder)
- Sección ventajas con iconos
- CTA final con botones

### 2. Simulador Vuelos
Página de producto con descripción detallada del módulo de vuelos, lista de características, misiones, avisos legales y botón de compra Hotmart (URL configurable)

### 3. Simulador Hoteles
Misma estructura que Vuelos, contenido adaptado a hoteles

### 4. Simulador Coches
Misma estructura que Vuelos, contenido adaptado a coches

### 5. Pack Simuladores
Página que presenta los 3 módulos juntos con botones individuales de compra, preparada para futuro pack conjunto

### 6. Sobre el Proyecto
Página con historia del creador, misión del proyecto y filosofía de formación práctica

### 7. Blog
Estructura lista con página de listado y página de artículo individual (contenido placeholder), preparada para SEO

### 8. Contacto
Formulario simple con campos email y mensaje, validación con feedback visual

## Navegación
- Header fijo con logo "Amadeus Survivor" estilo terminal y menú de navegación a todas las páginas
- Footer con links, copyright y enlaces útiles
- Menú hamburguesa en móvil

## SEO
- Meta tags y títulos optimizados con keywords (simulador amadeus, curso amadeus, gds training, etc.)
- Estructura semántica HTML (h1, h2, article, section)
- React Helmet para gestión de meta tags por página

## Botones Hotmart
- Todos los botones de compra son enlaces externos configurables (placeholder URLs editables)
- No se procesan pagos en la web

## Responsive
- Diseño adaptado a móvil, tablet y desktop
